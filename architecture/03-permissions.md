# Permissões e Segurança

## CONSUMIDOR

Pode:
- Ver produtos, preços, promoções e estabelecimentos.
- Criar listas de compras.
- Comprar ofertas pagas.
- Usar vouchers disponíveis conforme regras.
- Avaliar estabelecimentos.
- Ver seus próprios pedidos e histórico.

Não pode:
- Alterar produtos.
- Alterar preços.
- Acessar dados de outros consumidores.
- Acessar painel lojista ou admin.

## LOJISTA

Pode:
- Criar e editar seus estabelecimentos.
- Criar e editar produtos próprios.
- Criar promoções próprias.
- Criar vouchers se for premium.
- Ver relatórios próprios.
- Ver documentos próprios.

Não pode:
- Alterar dados de outros lojistas.
- Aprovar documentos.
- Acessar dados sensíveis de consumidores.

## ADMIN

Pode:
- Aprovar lojistas.
- Validar documentos.
- Gerenciar usuários.
- Gerenciar produtos, promoções, vouchers e ofertas.
- Acessar relatórios globais.
