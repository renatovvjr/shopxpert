# Roadmap MVP - ShopXpert

## Fase 1 - Base

- Estrutura do projeto.
- Supabase.
- Auth.
- Perfis: consumidor, lojista, admin.
- Documentação técnica.

## Fase 2 - Consumidor

- Cadastro/login.
- Home.
- Busca de produtos.
- Lista de compras.
- Comparativo de preços.
- Promoções e cupons.

## Fase 3 - Lojista

- Cadastro lojista.
- Upload de documentos.
- Dashboard.
- Produtos.
- Promoções.

## Fase 4 - Monetização

- Plano premium consumidor.
- Plano premium lojista.
- Vouchers.
- Ofertas pagas.
- Pedidos com retirada.

## Fase 5 - Admin

- Aprovação de lojistas.
- Validação de documentos.
- Gestão geral.
- Relatórios.

## Fase 6 - Expansão

- API para integração de preços.
- Entregas.
- Cashback.
- Recomendação inteligente.
