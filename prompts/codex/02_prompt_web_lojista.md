# Prompt Codex - Web Lojista

Gere o portal web do lojista em `apps/web-lojista` usando Next.js, TypeScript e Tailwind.

## Telas

- Boas-vindas lojista
- Login lojista
- Cadastro lojista
- Upload de documentos
- Dashboard
- Produtos
- Promoções
- Relatórios
- Perfil da loja
- Plano Premium
- Gestão de Vouchers

## Regras

- Cadastro deve permitir validação via GOV.BR ou envio de documentos.
- Upload de RG frente e verso, PDF do CNPJ e comprovante de residência.
- Se houver sócio, solicitar documentos do sócio.
- Mostrar aviso: validação manual pode levar até 4 dias úteis.
- Criar layout responsivo.
- Usar sidebar no dashboard.
