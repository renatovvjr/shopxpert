# Prompt Codex - Mobile Consumidor

Gere o app mobile consumidor em `apps/mobile` usando React Native, Expo e TypeScript.

## Telas

- Seleção de Perfil
- Boas-vindas
- Cadastro
- Login
- Home
- Lista de Compras
- Comparativo de Estabelecimentos
- Promoções e Cupons
- Conta/Perfil
- Avaliação
- Plano Premium
- Histórico de Vouchers
- Ofertas Pagas
- Pedido Confirmado
- Meus Pedidos

## Regras UX

- Usar laranja #F25022 para CTA.
- Usar verde #007E33 para economia, sucesso e loja.
- Fundo #FFFFFF.
- Inputs #FAFAFA com borda #CCCCCC.
- Botões com 50px de altura e raio 10px.
- Tipografia inicial: Arial/System.
- Tab bar inferior com Home, Listas, Histórico e Conta.
