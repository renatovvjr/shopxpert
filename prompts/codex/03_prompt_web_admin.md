# Prompt Codex - Web Admin

Gere o painel administrativo em `apps/web-admin` usando Next.js, TypeScript e Tailwind.

## Telas

- Login Admin
- Dashboard Admin
- Lojistas Pendentes
- Validação de Documentos
- Usuários
- Estabelecimentos
- Produtos
- Promoções
- Vouchers
- Ofertas Pagas
- Pedidos
- Relatórios
- Configurações

## Regras

- Admin aprova ou reprova lojistas.
- Admin visualiza documentos.
- Admin gerencia status de usuários e conteúdo.
- Criar tabelas, filtros, status badges e actions.
