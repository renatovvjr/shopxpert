# Prompt Supabase - RLS

Crie políticas de Row Level Security para:

- CONSUMIDOR
- LOJISTA
- ADMIN

Regras:
- Consumidor só acessa seus dados pessoais, listas, pedidos, vouchers e avaliações.
- Consumidor pode ler produtos, promoções e estabelecimentos ativos.
- Lojista só gerencia seus estabelecimentos, produtos, promoções, vouchers e relatórios.
- Admin tem acesso total controlado.
